Absolute and Relative URLs

Absolute URLs

Absolute URLs include the full web address:

  • Google
  • Wikipedia
  • GitHub

Relative URLs

Relative URLs are relative to the current page's location:

  • About Us (links to about.html in the same directory)
  • Contact Us (links to contact.html in the same directory)
  • Home (links to the parent directory's index.html)
  • View our Photo (links to a photo in the images folder)

Example Structure

Assuming the following directory structure:

    / (root directory)
    ├── index.html
    ├── about.html
    ├── contact.html
    └── images
        └── photo.jpg
