What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages and web applications.

What is CSS?

Cascading Style Sheets (CSS) is a stylesheet language used for describing the presentation of a document written in HTML or XML.

What is JavaScript?

JavaScript is a high-level programming language that enables interactive web pages and is an essential part of web applications.
